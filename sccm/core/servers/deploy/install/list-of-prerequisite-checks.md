# <a name="list-of-prerequisite-checks-for-system-center-configuration-manager"></a>Lista de verificações de pré-requisitos para o System Center Configuration Manager

*Aplica-se a: O System Center Configuration Manager (ramo atual)*

As secções seguintes descrevem pormenorizadamente as verificações de pré-requisitos disponíveis.

Para obter informações sobre a utilização do Verificador de pré-requisitos, consulte [Verificador de pré-requisitos](prerequisite-checker.md).  

##  <a name="BKMK_Security"></a>Verificações de pré-requisitos de direitos de segurança  
A tabela seguinte lista verifica se o Verificador de pré-requisitos efetua aos direitos de segurança de.

|Verificação de|EXPLICAÇÃO|Gravidade|Aplicabilidade de site|
|---|---|---|---|
|**Direitos de administrador no site de administração central**|Verifica a conta de utilizador que executa a configuração do Configuration Manager possui **administrador** direitos no computador do site de administração central. |Erro|Site primário|
|**Direitos administrativos no site primário de expansão**|Verifica se a conta de utilizador que executa a configuração tem **administrador** direitos no site primário autónomo que será expandido.|Erro|Site de administração central|
|**Direitos administrativos no sistema de sites**|Verifica se a conta de utilizador que executa a configuração do Configuration Manager possui **administrador** direitos no computador do servidor do site. |Erro|Site de administração central <br>Site primário, <br>Site Secundário|
|**Direitos administrativos da máquina CAS no site primário de expansão**|Verifica se a conta de computador do site de administração central tem **administrador** direitos no site primário autónomo que será expandido.|Erro|Site de administração central|
|**Ligação ao SQL Server no site de administração central**|Verifica se a conta de utilizador que executa a configuração do Configuration Manager no site primário para adesão a uma hierarquia existente possui o **sysadmin** função na instância do SQL Server para o site de administração central.|Erro|Site primário|
|**Direitos de conta de computador servidor do site administrativo**|Verifica se a conta de computador do servidor de site possui **administrador** direitos no servidor SQL e gestão de computadores do ponto.|Erro|Site primário, <br>SQL Server|
|**Sistema de sites para comunicação do SQL Server**| Verifica se um serviço Principal Name (SPN) válido está registado nos serviços de domínio do Active Directory para a conta configurada para executar o serviço do SQL Server para a instância do SQL Server que aloja o Gestor de configuração da base de dados do site. Um SPN válido tem de estar registado nos Serviços de Domínio do Active Directory para suportar a autenticação Kerberos.|Aviso|Site secundário, <br>Ponto de gestão|
|**Modo de segurança do SQL Server**|Verifica se o SQL Server está configurado para segurança de autenticação do Windows.|Aviso|SQL Server|
|**Direitos sysadmin do SQL Server**|Verifica se a conta de utilizador que executa a configuração do Configuration Manager possui os **sysadmin** função na instância do SQL Server selecionada para a instalação de base de dados do site. Esta verificação também falha quando o Programa de Configuração não consegue aceder à instância do SQL Server para verificar as permissões.|Erro|SQL Server|
|**Direitos de administrador do sistema do SQL Server para o site de referência**|Verifica se a conta de utilizador que executa a configuração do Configuration Manager possui os **sysadmin** função na instância de função do SQL Server selecionada como a base de dados do site de referência. SQL Server **sysadmin** são necessárias permissões de função para modificar a base de dados do site.|Erro|SQL Server|

|**Instância dedicada do SQL Server**|Verifica se uma instância dedicada do SQL server está configurada para alojar a base de dados do site do Configuration Manager. Em alternativa, pode desinstalar o outro site ou mover a base de dados para uma instância diferente para o SQL server.|Erro|Site de administração central <br>Site primário, <br>Site Secundário|
|**Componentes de servidor do Configuration Manager existentes no servidor**|Verifica se um servidor do site ou função do sistema de sites não já está instalada no computador selecionado para instalação do site.|Erro|Site de administração central <br>Site primário, <br>Site Secundário|
|**Exceção de firewall para SQL Server**|Verifica se a Firewall do Windows está desativada ou se uma exceção relevante da Firewall do Windows existe para o SQL Server. |**Exceção de firewall para SQL Server (site primário autónomo)**|Verifica se a Firewall do Windows está desativada ou se uma exceção relevante da Firewall do Windows existe para o SQL Server. Por predefinição, o SQL Server escuta na porta TCP 1433 e o SSB utiliza a porta TCP 4022.|Aviso|Site primário (apenas autónomo)|
|**Exceção de firewall para SQL Server para o ponto de gestão**|Verifica se a Firewall do Windows está desativada ou se uma exceção relevante da Firewall do Windows existe para o SQL Server.|Aviso|Ponto de gestão|
|**Configuração IIS HTTPS**|Verifica os enlaces de site do IIS para o protocolo de comunicação HTTPS. |**Execução do serviço IIS**|Verifica o que IIS está instalado e em execução no computador para instalar o ponto de gestão ou ponto de distribuição.|Erro|Ponto de gestão <br> Ponto de distribuição|
|**Corresponder agrupamento do site primário de expansão**|Verifica se a base de dados do site para o site primário autónomo que irá expandir tem o mesmo agrupamento que a base de dados do site no site de administração central.|Erro|Site de administração central|
|**Biblioteca Microsoft diferencial compressão remota (RDC) registada**|Verifica se a biblioteca RDC está registada no servidor de site do Configuration Manager.|Erro|Site de administração central <br>Site primário, <br>Site Secundário|
|**Microsoft Windows Installer**|Verifica a versão do Windows Installer. |**Microsoft XML Core Services 6.0 (MSXML60)**|Verifica se o MSXML 6.0 ou uma versão posterior está instalada no computador.|Aviso|Site de administração central <br>Site primário, <br>Site secundário, <br>Consola do Configuration Manager, <br>Ponto de gestão <br>Ponto de distribuição|
